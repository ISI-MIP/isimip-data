Instead of using CDO to select a bounding box, the cut-out can also be performed using ncks. This operation has a much better performance when applied to the high resolution data from CHELSA-W5E5 v1.0: W5E5 v1.0 downscaled with CHELSA v2.0>.

The extraction is performed using the command: ncks -O -h -d lat,SOUTH,NORTH -d lon,WEST,EAST IFILE OFILE
