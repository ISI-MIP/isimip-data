You can upload a custom NetCDF mask which needs to contain a mask variable var, which contains only zeros and ones. You can also set a name for the mask file on the server (which does not need to match the name of the uploaded file).

The files are then masked using the command: cdo -f nc4c -z zip_5 -ifthen -selname,VAR MASKFILE IFILE OFILE.

Everything outside is set to missing_value and the grid is kept.
