You can upload a custom vector mask in Shapefile or GeoJSON format, which will be converted to a NetCDF mask on the server. You can set the layer of the uploaded file to be used as mask.

The files are then masked using the command: cdo -f nc4c -z zip_5 -ifthen -selname,VAR MASKFILE IFILE OFILE.

Everything outside is set to missing_value and the grid is kept.
